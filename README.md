## About The Project

Poltergeist is a Bluetooth HID controller that transforms your Android device into a multi-purpose remote. By emulating standard keyboard and mouse profiles (HID), it enables wireless control of computers and other devices without requiring additional drivers or helper software.

### Key Features

- **Universal Bluetooth HID**: Control any device (Windows, Linux) as a standard wireless keyboard and mouse.
- **Advanced Input Support**: Full keyboard layout including Function (Fn) keys, arrow keys, and modifier combos.
- **Remote Wake & Management**: Wake connected devices from sleep and perform administrative tasks remotely.
- **Security Research Tool**: Use with caution—includes capabilities for rapid keystroke injection and payload delivery. **For educational and authorized testing scenarios only.**
- **Touchpad Interface**: Intuitive mobile interface for precise cursor movement and scrolling.

## Installation

> ⚠️ **Warning:**  
> Poltergeist is a tool intended for educational and authorized security research only.  
> **Do not use this software for unauthorized access or malicious purposes.**  
> Misuse may violate laws and result in severe penalties.

> ⚠️ **Under Development:**  
> **ESP32 hardware is required to use Poltergeist.**  
> At this time, I have not yet released the firmware code or setup instructions for the ESP32 device. Please stay tuned for updates as I work to make these resources available.

You can download the latest Android APK from the [Releases Page](https://github.com/Deadbush225/Poltergeist/releases).
> For most android the `poltergeist-arm64-v8a-release.apk` is enough 

1. Download the `.apk` file to your device.
2. Tap the file to install (you may need to allow installation from unknown sources).
3. Open ePrint and start managing your print jobs!

## Development

If you want to build the app yourself or contribute:

### Prerequisites

- [Dart SDK](https://dart.dev/get-dart)
- [Flutter SDK](https://flutter.dev/docs/get-started/install)

### Setup

```bash
# Clone the repository
git clone https://github.com/Deadbush225/Poltergeist.git

# Navigate to the project directory
cd ePrint

# Install dependencies
flutter pub get

# Run the app
flutter run
```
